# HemaNurseManager

A comprehensive management suite for Hematology/Oncology Nursing. This repository contains tools to assist with chemotherapy administration, protocol management, and checking for drug interactions.

## Modules

This project is divided into several specialized managers:

- **[Chemo Protocol Manager](./chemoProtocolManager)**: 
  - Manage and view chemotherapy protocols.
  - Calculate dosages and verify patient parameters.

- **[Chemo Medication Manager](./chemoMedicationManager)**: 
  - Detailed information on chemotherapy medications.
  - Administration guidelines and safety checks.

- **[Drug Interaction Manager](./drugInteractionManager)**: 
  - Check for compatibilities and interactions between IV medications.
  - Monitor Y-site and admixture compatibility.

## Usage

Each module is a self-contained web application. You can navigate to the respective folders and open the `index.html` file to run them locally.
